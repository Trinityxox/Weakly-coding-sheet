There are some specific steps to make a pizza. These are the steps in order.
  1. make pizza base
  2. add the tomato sauce
  3. add the toppings
  4. add cheese
  5. bake
  6. cut
  7. serve
The toppings are different, they change for each person. There is no order for toppings.
